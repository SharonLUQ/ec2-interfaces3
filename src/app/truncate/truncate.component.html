Ejemplo de uso del Pipe Truncate

Texto para ejemplificar el funcionamiento:

La programación en Angular es una de las tecnologías más populares para desarrollar aplicaciones web modernas.

Texto Truncado (20 caracteres):

{{'La programación en Angular es una de las tecnologías más populares para desarrollar aplicaciones web modernas.' | truncate:20}}

Texto Truncado (30 caracteres):

{{'La programación en Angular es una de las tecnologías más populares para desarrollar aplicaciones web modernas.' | truncate:30}}

Texto Truncado (40 caracteres):

{{'La programación en Angular es una de las tecnologías más populares para desarrollar aplicaciones web modernas.' | truncate:40}}

Texto Truncado (50 caracteres):

{{'La programación en Angular es una de las tecnologías más populares para desarrollar aplicaciones web modernas.' | truncate:50}}
